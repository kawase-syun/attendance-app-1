## 🆘 トラブルシューティング

### よくあるエラーと解決方法

**エラー: "GAS_URL is not defined"**
→ `app.js` の `CONFIG.GAS_URL` を設定してください

**LINEに通知が届かない**
→ GASのスクリプトプロパティを確認してください

**スプレッドシートに記録されない**
→ GASのデプロイ設定で「全員」を選択してください

**PWAがインストールできない**
→ アイコンファイル（icon-192.png, icon-512.png）が存在するか確認してください

**詳細なトラブルシューティング:**
→ `SETUP_GUIDE.md` を参照

**ブラウザの開発者ツール:**
```

```
Apps Scriptエディタ → 実行ログ
Logger.log() の出力を確認

### テストの進め方

1. **まず出勤打刻だけテスト**
   - LINEに通知が届くか
   - スプレッドシートに記録されるか

2. **次に退勤打刻をテスト**
   - 勤務時間が正しく計算されるか
   - LINEに通知が届くか

3. **最後に課題完了報告**
